# 살집 허브 페이지 우선순위 & 1주차 범위

- 작성일: 2026-03-16
- 목적: Top-tier 기준 허브(홈) 핵심 기능을 정의하고 1주차 최소 범위를 확정
- 원칙: 데이터 기반공사(coverage/accuracy/freshness) 우선, 허브는 최소 기능으로 빠르게 검증

## 허브 기능 우선순위 (Top-tier 기준 구체 스펙)

### P0: 허브 핵심 (검색 중심 구조)
- 검색 진입점 1순위: 지역/단지명/키워드 검색 바 + 지도 바운드 연동
- 결과 요약: 결과 개수, 최근 업데이트 시각, 데이터 출처 라벨
- 핵심 정렬/필터: 최신순, 가격순(상/하), 거래수 기준
- 신뢰성 표기: 출처(국토부 실거래가), 데이터 기준일
- 대표 카드: 검색 결과 리스트 + 대표 가격/최근 거래일/면적
- 상세 진입: 리스트 → 단지 상세로 이동

### P1: 성과 가속 (시장 스냅샷)
- 트렌드 모듈: 인기 단지/검색 급상승 키워드 (완료: 2026-03-19)
- 최근 거래 스냅샷: 최근 거래 N건/최근 가격 변화 요약 (완료: 2026-03-19)
- 지오 퀄리티 라벨: 정확/근사 좌표 구분 표시 (완료: 2026-03-19)
- 허브 상단 KPI: 전체 단지 수, 최근 3개월 거래 수(요약) (완료: 2026-03-19)

### P2: 확장 (추천/연계)
- 조건 기반 추천: 관심지역/가격대 기반 추천 리스트
- 금융 연계: 대출/금리 안내 섹션
- 생활 인프라: 학군/상권/교통 요약 블록

## Top-tier 패턴 → 허브 P0/P1 매핑 (패턴만 적용)

| 패턴 | 핵심 목적 | 허브 적용 위치 | 우선순위 |
| --- | --- | --- | --- |
| 지도 중심 검색 + 필터/정렬 | 빠른 탐색/발견 | 검색 바 + 지도 바운드 + 정렬 4종 | P0 |
| 신뢰 라벨(출처/업데이트 시각) | 데이터 신뢰 | 결과 요약 + 상세 상단 | P0 |
| 최근 거래/가격 변화 요약 | 시장 스냅샷 | 허브 스냅샷 모듈 | P1 |
| 인기 단지/급상승 키워드 | 재방문 유도 | 트렌드 모듈 | P1 |
| 평당가/갭가격 지표 | 가격 이해 | 카드/스냅샷 보조 지표 | P1 |
| 단지 비교(2개 비교) | 의사결정 | P1 이후 실험(별도 화면) | P1 |
| 학군/상권/생활 인프라 | 맥락 제공 | P2 섹션 | P2 |
| 관심등록/알림/문의 CTA | 전환 | 상세 CTA 고정 | P0 (상세) |

## 1주차 최소 범위 (현재 데이터/기능 기준)

### 포함 (반드시 구현)
- 검색 바 + 지도 연동
- 결과 요약(건수/최종 업데이트/출처 라벨)
- 정렬 4종: latest, price_desc, price_asc, deal_count
- 결과 리스트 카드(단지명/지역/최근 거래가/최근 거래일)
- 상세 이동(단지 상세 페이지 연결)

### 제외 (1주차에 하지 않음)
- 지역/브랜드/테마 랜딩(서브 디렉토리)
- 금융/추천/생활 인프라 모듈
- 고급 지도 분석/복잡한 비교 기능

## 완료 기준 (1주차)
- 홈에서 검색 → 결과 → 상세 이동이 끊기지 않음
- 최소 20개 키워드 샘플에서 0건 비율이 줄어듦

- 최근 업데이트 시각/출처가 허브와 상세에 표기됨
