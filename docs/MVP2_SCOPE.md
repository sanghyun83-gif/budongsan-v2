# MVP2_SCOPE

## 포함 범위 (MVP-2 In Scope)
1. 검색 UI + 지도 탐색 루프
2. 단지 상세 페이지(기본형)
3. `/api/complexes/:id`, `/api/complexes/:id/deals`
4. 최소 ingest/transform 스크립트
5. 신뢰성 라벨(데이터 출처, 최종 업데이트 시각)

## 제외 범위 (MVP-2 Out of Scope)
1. 중개사 KYC 운영
2. 허위매물 제재 전체 workflow
3. AI 대화형 검색
4. 3D/VR 투어
5. 수익화 상품(광고/결제)

## 메모
1. 관심등록/알림 CTA는 placeholder 허용
2. 부분 완료 모듈은 feature flag로 운영 가능
